## การเล่น

หลังจากกดปุ่มเริ่มเกมเเล้วจะสามารถเล่นได้โดยจะมีการควยคุมนกได้โดย
W==บินขึ้น
A==บินไปซ้าย
S==บินลงล่าง
D==บินไปขวา
โดยการเล่นจะต้องบินหลบสิ่งกีดขวางที่มาจากทางด้านข้าง
เเละจะมีคะเเนนScores เพื่อบอกเเต้มที่คุณได้เก็บเหรียญไว้
